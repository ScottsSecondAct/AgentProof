# Contributing to AgentProof

Thanks for your interest in AgentProof!

This project is currently in **early development** and is **not yet open to external contributions**. The architecture and roadmap are still being actively shaped, and accepting contributions at this stage would create coordination overhead that slows things down.

## What You Can Do Now

- **Report bugs** — If you find a bug, please [open an issue](https://github.com/ScottsSecondAct/AgentProof/issues/new?template=bug_report.md). Include your OS, Rust/Python version, and steps to reproduce.
- **Request features** — Have an idea? Open an issue with the `enhancement` label. No guarantees, but good ideas get noticed.
- **Ask questions** — If something in the docs is unclear, an issue is fine for that too.

## When Will Contributions Be Accepted?

The plan is to open contributions once the project reaches a stable architecture and has clear contribution guidelines in place. When that happens, this document will be updated with:

- A guide to setting up the development environment
- Code style and commit conventions
- The pull request and review process
- Which areas of the codebase are ready for outside help

## In the Meantime

Feel free to fork the repo and experiment. If you build something interesting on top of AgentProof, share it in the Discussions tab.
